# Contributing to CosmoBond Watch Face

Thank you for your interest in contributing! Please follow these guidelines:

1. Discuss significant changes via issue or proposal before starting work.
2. Follow the roadmap tasks in `AGENTS.md`, claiming tasks before you begin.
3. Keep each pull request focused on a single checklist item and include evidence of passing tests.
4. Run the documented linting and testing commands locally before opening a pull request.
5. Ensure documentation updates accompany behavior or policy changes.

By contributing, you agree that your submissions will be licensed under the repository's license.
