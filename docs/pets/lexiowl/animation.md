# CosmoBond LexiOwl Animation Plan — Gemini GIF Workflow

## Production Approach
- **Generator:** Google Gemini video diffusion leveraging crisp cel shading with floating typographic elements.
- **Export guardrails:** 438 × 438 px, 4-second loops, parchment navy (#080B14) background, and steady camera for legible flashcards.
- **Post-processing:** Convert to 60 fps PNG stacks, vectorize key type glyphs in Illustrator, and export GIF/WebM plus sprite sheets.

## Pet Model Snapshot
- Scholarly owl with heart-shaped face, layered feathers resembling stacked book pages, and quill-tipped wings.
- Palette: indigo body, cream face, and amber feather edges with metallic gold accents.
- Face: oversized round glasses, thoughtful amber eyes, and a tiny beak; art style nods to storybook etchings.
- Accessories: floating dictionary tome, bookmark tassels, and letter tiles orbiting the wings.
- Perches atop a brass armillary sphere occupying the dial center.

## Interaction Map
- **Lesson Focus:** Triggered when the user starts a language lesson or flashcard drill, emphasizing attentive study motion.
- **Quiz Success:** Fires when the user answers correctly or logs progress, highlighting celebratory flips.

## Animation States

### Idle State — "Armillary Perch"
- **Title:** Armillary Perch
- **Pet Visual Description:**
  - LexiOwl perches calmly with wings folded; feathers shimmer softly.
  - Glasses glint; eyes blink slowly.
  - Armillary rings rotate gently with subtle reflections.
  - Letter tiles drift lazily, forming faint words.
  - Bookmark tassel sways.
- **Action Sequence:**
  1. Owl adjusts glasses with one wing tip, eyes focusing.
  2. It ruffles feathers, sending dust motes into the air.
  3. Armillary ring spins a quarter turn; letter tiles rearrange.
  4. Owl settles back, closing eyes momentarily.

### Interaction State — Lesson Focus "Lexicon Beam"
- **Title:** Lexicon Beam
- **Pet Visual Description:**
  - Feathers glow indigo; glasses project holographic syllables.
  - Wings extend, quill tips shining.
  - Dictionary tome opens, pages flipping rapidly.
  - Letter tiles align into sentences.
  - Background darkens to spotlight the owl.
- **Action Sequence:**
  1. LexiOwl spreads wings, summoning a ribbon of glowing text.
  2. It traces the ribbon with a quill feather, highlighting syllables.
  3. The owl nods approvingly, eyes following the line.
  4. Pages snap shut gently, resetting for the next lesson.

### Interaction State — Quiz Success "Feather Flip"
- **Title:** Feather Flip
- **Pet Visual Description:**
  - Palette brightens with gold sparkles.
  - Owl flips upside down mid-air; glasses stay perfectly aligned.
  - Armillary rings explode with celebratory symbols (checkmarks, stars).
  - Dictionary closes, acting as a springboard.
  - Tail feathers fan out like a ribbon.
- **Action Sequence:**
  1. LexiOwl launches upward, wings tucked.
  2. Mid-air, it flips, scattering gold letters.
  3. The owl lands on the tome, which compresses and rebounds.
  4. It bows with wings outstretched before looping.

### Sick State — "Missed Session"
- **Title:** Missed Session
- **Pet Visual Description:**
  - Feathers dull; glasses slide down the beak.
  - Eyes droop; armillary slows almost to a stop.
  - Letter tiles fall to the base, forming jumbled piles.
  - Tome closes with a sigh.
  - Owl hunches, wings wrapping around itself.
- **Action Sequence:**
  1. LexiOwl yawns, rubbing eyes with a wing.
  2. It tries to open the tome, but the pages refuse.
  3. The owl slumps, feathers ruffled.
  4. Tiles tumble again, prompting another weary sigh.

### Happy State — "Polyglot Waltz"
- **Title:** Polyglot Waltz
- **Pet Visual Description:**
  - Feathers shimmer with shifting gradients representing different languages.
  - Glasses sparkle; eyes joyful.
  - Armillary rings spin in choreography; letter tiles dance around the owl.
  - Tome floats, projecting translations.
  - Owl waltzes mid-air with gentle twirls.
- **Action Sequence:**
  1. LexiOwl hops off the sphere, gliding gracefully.
  2. It twirls, wings tracing elegant calligraphy strokes.
  3. Translations flash around the body as it laughs.
  4. The owl lands lightly, ready to dance again.

### Runaway State — "Library Fade"
- **Title:** Library Fade
- **Pet Visual Description:**
  - Body becomes semi-transparent parchment; feathers turn to fluttering pages.
  - Glasses fall slowly toward the viewer.
  - Armillary dissolves into floating ink.
  - A doorway shaped like a library shelf opens behind.
  - Letter tiles swirl into a whirlwind.
- **Action Sequence:**
  1. LexiOwl looks back wistfully, clutching the tome.
  2. It steps backward into the shelf doorway, pages peeling away.
  3. Glasses drift down, caught by invisible wind.
  4. Doorway closes, leaving only swirling letters before reset.

### Death State — "Gilded Volume"
- **Title:** Gilded Volume
- **Pet Visual Description:**
  - Owl transforms into a golden embossed relief on the tome cover.
  - Glasses rest atop the book; wings etched into the surface.
  - Armillary rings collapse into a bookmark clasp.
  - Soft light emanates from engraved text.
  - Letter tiles arrange into "Forever Learning".
- **Action Sequence:**
  1. LexiOwl exhales, feathers turning to gold dust.
  2. Dust settles onto the tome, forming the relief.
  3. Bookmark clasp locks with a gentle click.
  4. Light pulses once through the title before the loop rewinds.
