# CosmoBond TranquiliTurtle Animation Plan — Gemini GIF Workflow

## Production Approach
- **Generator:** Google Gemini video diffusion emphasizing soft volumetric lighting and meditative water ripples.
- **Export guardrails:** 438 × 438 px, 4-second loops, indigo pond backdrop (#040912), and subtle camera dolly to follow breathing.
- **Post-processing:** Convert to 60 fps PNG stacks, enhance aura gradients in Procreate, and assemble GIF/WebM plus sprite sheets.

## Pet Model Snapshot
- Gentle turtle with jade-green shell etched with lotus motifs, semi-transparent fins, and floating prayer beads.
- Face: closed-lid eyes with long lashes, serene smile, and blush along the cheeks.
- Accessories: lotus cushion, hovering incense orb, and breathing meter ring.
- Hovers mid-frame, shell occupying the lower half of the dial.

## Interaction Map
- **Meditation Pose:** Triggered when the user launches a guided breathing session, showcasing floating lotus poses.
- **Breath Coaching:** Fires during quick stress relief prompts, highlighting animated breathing rings.

## Animation States

### Idle State — "Calm Drift"
- **Title:** Calm Drift
- **Pet Visual Description:**
  - TranquiliTurtle floats on a lotus cushion; shell glows softly.
  - Incense orb releases slow spirals.
  - Prayer beads orbit wrists.
  - Eyes half-closed; smile relaxed.
  - Background ripples gently.
- **Action Sequence:**
  1. Turtle inhales, shell expanding subtly.
  2. Cushion sinks slightly, releasing ripples.
  3. Exhale sends glitter particles upward.
  4. Turtle tilts head, returning to neutral.

### Interaction State — Meditation Pose "Lotus Hover"
- **Title:** Lotus Hover
- **Pet Visual Description:**
  - Shell emits bright jade aura; fins fold into lotus mudras.
  - Eyes close fully; beads align above the shell.
  - Cushion levitates higher.
  - Breathing ring displays inhale/hold/exhale cues.
  - Background darkens to spotlight the turtle.
- **Action Sequence:**
  1. Turtle raises into mid-air, legs crossing in meditation.
  2. It inhales, breathing ring expanding outward.
  3. A hold moment glows golden around the shell.
  4. Exhale lowers the turtle gently before looping.

### Interaction State — Breath Coaching "Aura Sweep"
- **Title:** Aura Sweep
- **Pet Visual Description:**
  - Shell glows turquoise; fins extend outward.
  - Eyes open softly, offering encouragement.
  - Breathing meter forms arcs along the shell edges.
  - Incense orb pulses faster.
  - Background ripples accelerate.
- **Action Sequence:**
  1. Turtle extends fins, tracing a glowing arc labeled "Breathe In".
  2. It brings fins inward at the hold cue, shell pulsing.
  3. Exhale arcs sweep outward, clearing stress icons.
  4. Turtle smiles gently as the arcs fade.

### Sick State — "Stress Warning"
- **Title:** Stress Warning
- **Pet Visual Description:**
  - Colors dull; shell closes partially.
  - Eyes open with worry; incense flickers erratically.
  - Beads tangle.
  - Cushion droops, nearly touching water.
  - Background ripples become jagged.
- **Action Sequence:**
  1. Turtle trembles, clutching beads.
  2. It breathes rapidly, shell shuddering.
  3. Incense orb flickers red, casting sharp shadows.
  4. Turtle curls into the shell before repeating.

### Happy State — "Lotus Bloom"
- **Title:** Lotus Bloom
- **Pet Visual Description:**
  - Shell blossoms into translucent lotus petals.
  - Eyes sparkle; smile wide.
  - Cushion erupts with luminous flowers.
  - Beads release glitter.
  - Background filled with aurora waves.
- **Action Sequence:**
  1. Turtle spins slowly, petals opening around it.
  2. It releases a joyful laugh, scattering petals outward.
  3. Turtle bows, pressing fins together in gratitude.
  4. Petals reassemble under the shell before the next spin.

### Runaway State — "Retreat Dive"
- **Title:** Retreat Dive
- **Pet Visual Description:**
  - Body becomes semi-transparent water; shell darkens.
  - Incense orb extinguishes.
  - Cushion dissolves into mist.
  - Eyes glance back sadly.
  - Background forms a whirlpool doorway.
- **Action Sequence:**
  1. Turtle looks back once, beads falling away.
  2. It dives into the whirlpool, body dissolving into water trails.
  3. Shell disappears beneath the surface.
  4. Whirlpool closes, leaving lingering ripples.

### Death State — "Still Pond"
- **Title:** Still Pond
- **Pet Visual Description:**
  - TranquiliTurtle rests beneath translucent water; shell becomes stone.
  - Lotus cushion unfurls into floating petals around the body.
  - Beads wrap the shell gently; eyes closed.
  - Incense orb transforms into a candle flame.
  - Pond surface mirrors the moon.
- **Action Sequence:**
  1. Water calms, revealing the resting turtle.
  2. Candle flame flickers once, illuminating the shell.
  3. Petals drift outward in slow circles.
  4. Moon reflection fades before the loop restarts.
